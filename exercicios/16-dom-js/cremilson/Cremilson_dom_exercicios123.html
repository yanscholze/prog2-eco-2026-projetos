    3 Perguntas e respostas

    O que é o DOM?

    O DOM (Document Object Model) é uma representação em árvore dos elementos HTML, que o navegador cria para que o JavaScript possa interagir com a página.

    Qual a diferença entre innerHTML e textContent?

    innerHTML interpreta e renderiza HTML dentro da string, enquanto textContent trata tudo como texto puro, sem processar tags.

    Para que serve addEventListener?

    addEventListener registra uma função (callback) que será executada quando um evento específico ocorrer em um elemento, como 'click', 'input' ou 'keydown'.
