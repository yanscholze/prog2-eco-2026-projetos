## Descrição da solução

Foi criada uma classe base chamada "Pagamento", que possui
o atributo valor e um método chamado processar().

A partir dela foram criadas três classes que herdam suas
características: "PagamentoCartao", "PagamentoPix" e
"PagamentoBoleto".

Cada uma dessas classes implementa o método processar()
de forma diferente para representar cada tipo de pagamento.

No final do código são criados exemplos de pagamentos
e uma função é usada para processá-los e mostrar os
resultados no console.
