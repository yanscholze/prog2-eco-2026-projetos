## Descrição da solução

Foi criada uma classe chamada "Livro" contendo título e autor.

Também foi criada a classe "UsuarioBiblioteca", que possui
uma lista privada de livros emprestados e métodos para
emprestar e listar livros.

Além disso, foi criada a classe "Biblioteca", responsável
por armazenar os usuários e listar todos os empréstimos
realizados.

No final do código são criados livros, um usuário e uma
biblioteca para testar o funcionamento e mostrar os
empréstimos no console.
