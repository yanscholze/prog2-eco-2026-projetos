## Descrição da solução

Foi criada uma classe chamada "CarrinhoDeCompras" que possui
uma lista privada de itens.

Foram implementados métodos para adicionar itens, remover itens,
listar os produtos do carrinho e calcular o valor total.

Também foram feitas validações para impedir preços ou quantidades
menores ou iguais a zero.

No final do código alguns itens são adicionados ao carrinho
para testar o funcionamento e mostrar os resultados no console.
