## Seção 1 – IDEs e editores de código
> Categoria obrigatória

**Ferramentas comparadas:**
- Ferramenta 1: `Visual Studio Code
- Ferramenta 2: `WebStorm`
- Ferramenta 3: `Sublime Text`
- Ferramenta 4: `Atom`

**Introdução:**
_IDE´s e editores de código são importantes no desenvolvimento web porque ajudam o programador a trabalhar mais rápido, com menos erros e mais organização._

**Tabela comparativa:**

| Característica | Visual Studio Code | WebStorm | Sublime Text | Atom |
|---|---|---|---|---|
| Facilidade de instalação |fácil |fácil |fácil | fácil|
| Suporte a JavaScript e Node.js | sim| sim| sim|sim,porém não é mantido oficialmente |
| Suporte a HTML e CSS |sim |sim | sim| sim,porém descontinuado|
| Debugger integrado |sim |sim |não |não|
| Terminal integrado |sim |sim |não|não |
| Integração com Git |sim |sim| não|não |
| Extensões disponíveis |sim |sim |sim | não|
| Performance |2º |3º |1º|4º |
| Custo / licença |gratuito | pago| pago|gratuito |
| **Nota geral (1 a 5)** |5 |4 |4 | 1|

**Ferramenta escolhida:** `VS Code`

**Justificativa:**
_preferimos utilizar Visual Studio Code por ter mais familiaridade e ser uma ferramenta gratuita que engloba todas as nossas necessidades._

## Seção 2 – Clientes de banco de dados
> Categoria obrigatória

**Ferramentas comparadas:**
- Ferramenta 1: `DBeaver`
- Ferramenta 2: `DataGrip`
- Ferramenta 3: `HeidiSQL`
- Ferramenta 4: `pgAdmin`

**Introdução:**
_Um cliente de banco de dados é um software que permite conectar, gerenciar e consultar bancos de dados de forma prática, sem precisar digitar comandos diretamente no servidor._

**Tabela comparativa:**

| Característica | DBeaver | DataGrip | HeidiSQL | pgAdmin |
|---|---|---|---|---|

| Visualização de tabelas |5 |5 |4 |3 |
| Editor de queries SQL | 5|5 |3 | 4|
| Integração com a IDE |5 |5 |3 |4 |
| Custo / licença |possui versão gratuita e paga |paga |gratuito | gratuito|
| **Nota geral (1 a 5)** |5 |3 |4 | 3|

**Ferramenta escolhida:** `Dbeaver`

**Justificativa:**
_É o com melhor nota dentre os pesquisados e possui versão gratuita. _

## Seção 3 – Ferramentas de teste de API
> Categoria obrigatória

**Ferramentas comparadas:**
- Ferramenta 1: `Postman`
- Ferramenta 2: `Insomnia`
- Ferramenta 3: `Swagger Inspector`
- Ferramenta 4: `SoapUI`

**Introdução:**
_As ferramentas de teste de API são essenciais no desenvolvimento moderno porque garantem que os sistemas se comuniquem corretamente e funcionem como esperado._

**Tabela comparativa:**

| Característica | Postman | Insomnia |Swagger Inspector | SoapUI |
|---|---|---|---|---|
| Facilidade de uso |5 |5 | 5| 2|
| Suporte a métodos HTTP (GET, POST, PUT, DELETE) |sim |sim |sim |sim |
| Visualização da resposta | 5|5 |4 | 3|
| Organização de requisições |5 |5 | 4|3 |
| Integração com a IDE |1 | 1| 1| 1|
| Custo / licença |possui versão gratuita e paga |gratuito | gratuito|possui versão gratuita e paga |
| **Nota geral (1 a 5)** |5 |5 |4 | 2|

**Ferramenta escolhida:** `Insomnia`

**Justificativa:**
_Postman e insomnia são quase equivalentes em qualidade, com a diferença que Insomina parece ter uma interface mais simples, limpa e mais leve, enquanto Postman é mais robusto e possui automação complexa.  _
