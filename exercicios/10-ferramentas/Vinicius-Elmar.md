## Seção 2 – Clientes de banco de dados
> Categoria obrigatória

**Ferramentas comparadas:**
- Ferramenta 1: `MySQL Workbench`
- Ferramenta 2: `MariaDB`
- Ferramenta 3: `DBeaver`
- Ferramenta 4: `TablePlus`

**Introdução:**
Clientes de banco de dados são ferramentas utilizadas para visualizar, gerenciar e manipular bancos de dados. Elas permitem visualizar tabelas, editar registros e administrar a estrutura do banco de dados de forma mais prática. Essas ferramentas são importantes em projetos de desenvolvimento web porque facilitam a interação entre a aplicação e o banco de dados.

| Característica ----------| Ferramenta 1 | Ferramenta 2 | Ferramenta 3 | Ferramenta 4 |

| **Nota geral (1 a 5)**   |     4.6      |      4.6     |     4.6      |      4       |

**Ferramenta escolhida:** `MySQL Workbench`

**Justificativa:**
O MySQL Workbench foi escolhido por ser uma ferramenta oficial desenvolvida para trabalhar com bancos de dados MySQL, oferecendo boa integração com esse sistema gerenciador. Ele possui uma interface gráfica intuitiva que facilita a criação, visualização e administração de bancos de dados, permitindo executar consultas SQL, modelar tabelas e gerenciar conexões de forma organizada.

## Seção 3 – Ferramentas de teste de API
> Categoria obrigatória

**Ferramentas comparadas:**
- Ferramenta 1: `Postman`
- Ferramenta 2: `Insomnia`
- Ferramenta 3: `Hoppscotch`
- Ferramenta 4: `ThunderClient`

**Introdução:**
Ferramentas de teste de API permitem enviar requisições HTTP para serviços web e visualizar suas respostas. Elas são utilizadas para testar endpoints, validar respostas de servidores e facilitar o desenvolvimento de APIs REST. Essas ferramentas ajudam os desenvolvedores a verificar se as rotas da aplicação estão funcionando corretamente.

**Tabela comparativa:**

| Característica             | Ferramenta 1 | Ferramenta 2 | Ferramenta 3 | Ferramenta 4 |
|----------------------------|--------------|--------------|--------------|--------------|
| Facilidade de uso          |      5       |       4      |      4       |      5       |
| Suporte a métodos HTTP     |      5       |       5      |      5       |      5       |
| Visualização da resposta   |      5       |       5      |      4       |      4       |
| Organização de requisições |      5       |       4      |      3       |      4       |
| Integração com a IDE       |      4       |       4      |      1       |      5       |
| Custo / licença            |      3       |       3      |      5       |      5       |
| **Nota geral (1 a 5)**     |     4.5      |      4.2     |     3.7      |     4.7      | 

**Ferramenta escolhida:** `Postman`

**Justificativa:**
O Postman é uma das ferramentas mais populares para teste de APIs. Ele permite organizar requisições em coleções, visualizar respostas de forma clara e testar diferentes métodos HTTP facilmente. Além disso, possui grande comunidade e documentação.

Essas ferramentas ajudam a garantir que o código desenvolvido siga boas práticas, seja mais fácil de manter e possa ser executado em ambientes padronizados.
