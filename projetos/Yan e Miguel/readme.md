# 🏎️ Sistema de Gerenciamento de Pista de Corrida

## 📄 Descrição do Projeto

O presente projeto tem como objetivo o desenvolvimento de um sistema de gerenciamento para uma pista de corrida, visando otimizar a organização, controle e operação das atividades realizadas no local. O sistema será responsável por integrar informações relacionadas a pilotos, veículos, corridas e utilização da pista, proporcionando maior eficiência, segurança e praticidade tanto para os usuários quanto para os administradores.

A plataforma permitirá o cadastro e gerenciamento de pilotos, incluindo seus dados pessoais e histórico de participação em corridas. Também será possível realizar o controle dos veículos utilizados, garantindo que estejam devidamente registrados e aptos para uso na pista.

Além disso, o sistema oferecerá funcionalidades de agendamento de corridas, permitindo verificar a disponibilidade da pista, organizar horários e evitar conflitos. Durante a execução das corridas, será possível registrar resultados, como tempos de volta e classificações, possibilitando posteriormente a consulta de desempenho e a geração de rankings.

O sistema contará com diferentes níveis de acesso, como administradores, funcionários e pilotos, cada um com permissões específicas. Os administradores terão controle total sobre o sistema, podendo gerenciar usuários, corridas e relatórios. Os funcionários atuarão na operação diária, realizando cadastros, agendamentos e registros. Já os pilotos poderão se cadastrar, agendar corridas e consultar seus resultados.

Por fim, o sistema busca oferecer uma solução completa para o gerenciamento de pistas de corrida, contribuindo para uma melhor experiência dos usuários, organização das atividades e apoio à tomada de decisões através de relatórios e dados consolidados.

## 💻 Tecnologias Utilizadas

Este projeto será desenvolvido utilizando um conjunto de tecnologias modernas voltadas para o desenvolvimento web, aplicando na prática os conceitos aprendidos na disciplina de Programação II.

### 🧠 Linguagens de Programação

- **JavaScript**  
  Utilizado como linguagem principal do projeto, responsável pela lógica do sistema, manipulação de dados, validações e interação com o usuário.

- **HTML**  
  Utilizado para a estruturação das páginas do sistema.

- **CSS**  
  Utilizado para a estilização da interface, garantindo melhor usabilidade e experiência do usuário.

---

### 🗄️ Banco de Dados

Será utilizado um banco de dados para armazenamento persistente das informações do sistema, como:

- Cadastro de pilotos  
- Registro de veículos  
- Agendamento de corridas  
- Resultados e tempos  

O uso de banco de dados permite maior organização, segurança e escalabilidade do sistema.

---

### ⚙️ Tecnologias de Back-end

O projeto contará com uma camada de back-end para processamento das regras de negócio e comunicação com o banco de dados.

- **Node.js**  
  Responsável por executar o código no lado do servidor, permitindo a criação de APIs e integração com o banco de dados.

---

### 🛠️ Ferramentas de Desenvolvimento

- **Visual Studio Code**  
  Editor de código utilizado no desenvolvimento do projeto, escolhido por ser leve, eficiente e possuir diversas extensões que auxiliam na produtividade.

- **Git**  
  Utilizado para controle de versão, permitindo o acompanhamento das alterações no código e facilitando o trabalho em equipe.

- **GitHub**  
  Plataforma utilizada para hospedagem do repositório do projeto, possibilitando colaboração entre os integrantes e gerenciamento do código.
